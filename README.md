## 摘要

> 本项目是基础wechaty实现基金相关信息提醒，主要包含基金的实时涨幅，财富网相关基金新闻，微信公众号相关大佬的文章等的推送，最后非常感谢wechaty的无私奉献。

## 包含的功能如下：

* 对已选的基金进行涨幅信息推送
* 财富网，北上资金的实时信息推送
* 每天推送公众号的一些大佬文章，金融相关的
* 每日天气推送
* 下载音乐
* 。。。。
